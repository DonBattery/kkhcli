# kkhcli

Simple admin CLI tool for KKHC APP

### Usage: kkhcli \[option\] \[username\]

```SHELL
Options:
  -a, --add string
    	Add user
  -d, --disable string
    	Disable user
  -e, --enable string
    	Enable user
  -l, --list
    	List users
  -r, --reset string
    	Reset password
  -s, --seed
    	Seed database
  -u, --url string
    	WEB URL (default "https://kkhc.eu/admin")
```
